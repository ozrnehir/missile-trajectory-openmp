# 🚀 Missile Trajectory Simulation with OpenMP (C++)

This project demonstrates **parallel computing in C++ using OpenMP** by simulating missile trajectories and analyzing performance improvements through multi-threading.

---

## 📌 Overview

The goal of this project is to compare **sequential vs parallel execution** and observe how parallelization affects performance in computational simulations.

---

## ⚙️ Compile & Run

Make sure you have a C++ compiler with OpenMP support.

### Compile:
```bash
g++ -fopenmp main.cpp -o simulation
```

### Run:
```bash
./simulation
```

## 📊 Key Insights

- More threads do **not always** mean better performance  
- Proper workload distribution is **critical**  
- Parallelization improves performance only in **certain conditions**

---

## 📌 Conclusion

Although parallel computing can significantly improve performance, it depends on factors such as:
- Task complexity
- Thread management
- Data distribution

This project highlights the importance of **efficient parallel design**.
